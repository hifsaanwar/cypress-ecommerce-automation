SauceDemo Cypress Automation Framework

This repository contains a fully implemented Cypress automation framework built for the SauceDemo
 application.

 
The project follows industry best practices, including:

1. Page Object Model (POM) architecture
2. Reusable components
3. Fixtures for test data
4. Clean test separation (Login / Inventory / Cart / Checkout / Side Menu)
5. Readable, maintainable test flow
6. Portfolio-ready structure
